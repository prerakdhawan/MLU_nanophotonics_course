# Theoretical and computational nanophotonics
This repository provides the packages and assigments for the tutorials on *Micro and Nanophotonics* lectures at MLU.

This tutorial series first gives a general overview of few numerical techniques like Transfer Matrix Method (TMM), Finite Element Method (FEM), Finite-Difference Time and Frequency Domain (FDTD and FDFD) and other methods often used by researchers for a prototypical wave-optics problem. At a later stage, it dives deeper into Finite-Difference Time-Domain method and using an open-source package [MEEP](https://meep.readthedocs.io/en/master/) to look at common wave-optics phenomena.

To get started with the assignment, use the `launch binder` icon above and look at the uploaded notebooks (`.ipynb` files). The assignments questions will also be uploaded here.
